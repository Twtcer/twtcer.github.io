基本使用

1、打开git bash命令窗口；

2、使用git checkout xxx进入主分支下（xxx为分支名称，如develop）；

使用git checkout xxx进入主分支下（xxx为分支名称，如develop）


3、使用git pull命令从主分支拉最新的代码； 

git pull

 




4、pull完成之后，再使用git checkout xxx进入自己的分支下，以免下次直接提交到develop



git checkout xxx
